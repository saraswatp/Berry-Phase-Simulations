# Berry Phase and Bloch Sphere Simulator

An interactive Python visualization tool for exploring Berry phase accumulation, spin dynamics, and semiclassical transport phenomena.  
Built using **Tkinter**, **Matplotlib**, and **NumPy**.

## Features
- Visualizes Bloch sphere precession and Berry curvature maps.
- Allows time-dependent modulation of magnetic field and mass terms.
- Exports simulation data to CSV.
- User-adjustable parameters via GUI.

## Requirements
Python 3.9+
